Portfolio 3 - Individual Differences Within AI Question Framing and Emotional Responses to Purpose Reflection in College Upperclassmen

Goal

In this project, I build off of my earlier portfolio piece to explore how negative affect varies across individual‑difference variables in my AI post‑graduation planning study, focusing on gender, class year, and age as a baseline descriptive step before I add psychological traits like openness in later work. With this, I want to practice building a clean analytic tibble from my SPSS export, create multiple ggplot visualizations of distress across groups, and add simple numeric summaries that connect directly to the figures. The original study recruited 60 U.S. juniors and seniors to report their post‑graduation plan clarity, respond to either two specific or two open‑ended AI‑generated purpose questions, and then rate their negative affect, with the main finding that higher plan clarity predicted lower distress while AI question type and its interaction with clarity were non‑significant.

Products

My products will include a cleaned analytic tibble that contains negative affect, gender, class year, and age, with gender and class year recoded into factor labels that are readable and removing any rows with values missing so the same complete‑case sample is used for all of the analyses. With this tibble, I created three visualizations: a plot of negative affect by gender, a plot of negative affect by class year, and a scatterplot of age versus negative affect with a smooth line to show any trend in distress across the narrow upper‑division age range. In addition to the graphs, I generated small summary tibbles reporting the mean and standard deviation of negative affect by gender and by class year, plus the overall mean, standard deviation, and range of negative affect for the full sample, so I can move between visual and numeric descriptions of the same patterns.

studydata_raw <- read_sav("AI Purpose Questions and Plan Clarity Study.sav") %>% 
  as_tibble()
studydata_emotion <- studydata_raw %>% 
  transmute(
    id           = ID,
    condition    = condition,         
    neg_affect   = NegAff_Mean
  )
gender_demo <- studydata_raw %>% 
  transmute(
    id     = ID,
    gender = Gender
  )

classyear_demo <- studydata_raw %>% 
  transmute(
    id         = ID,
    class_year = Study_Year
  )

age_demo <- studydata_raw %>% 
  transmute(
    id  = ID,
    age = Age
  )
studydata_emotion_demo <- studydata_emotion %>% 
  left_join(gender_demo,    by = "id") %>% 
  left_join(classyear_demo, by = "id") %>% 
  left_join(age_demo,       by = "id")
overall_distress_summary <- studydata_emotion_demo %>% 
  summarise(
    n        = n(),
    mean     = mean(neg_affect),
    sd       = sd(neg_affect),
    min      = min(neg_affect),
    max      = max(neg_affect)
  )
gender_distress_summary <- studydata_emotion_demo %>% 
  group_by(gender) %>% 
  summarise(
    n    = n(),
    mean = mean(neg_affect),
    sd   = sd(neg_affect),
    .groups = "drop"
  )
classyear_distress_summary <- studydata_emotion_demo %>% 
  group_by(class_year) %>% 
  summarise(
    n    = n(),
    mean = mean(neg_affect),
    sd   = sd(neg_affect),
    .groups = "drop"
  )
combined_distress_summary <- bind_rows(
  overall_distress_summary %>% 
    mutate(
      grouping = "overall",
      level    = "all"
    ),
  gender_distress_summary %>% 
    mutate(
      grouping = "gender",
      level    = as.character(gender)
    ) %>% 
    select(grouping, level, n, mean, sd),
  classyear_distress_summary %>% 
    mutate(
      grouping = "class_year",
      level    = as.character(class_year)
    ) %>% 
    select(grouping, level, n, mean, sd)
) %>% 
  relocate(grouping, level)
