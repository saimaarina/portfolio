Portfolio 6 - Narcissiscm and Extraversion as Predictors

Goal

For my Research Methods 1 class, I wrote a paper looking at narcissism and extraversion as predictors of status seeking and group affiliation behaviors using the same mTurk data. In this portfolio piece, I want to make visualizations/figures of my main findings in order to demonstrate them in a visually appealing way that is not found in the original paper that I wrote. It will illustrate how narcissism and extraversion relate to group‑affiliation and status‑seeking motives in a sample of 300 adults, emphasizing that narcissism uniquely predicts status seeking whereas extraversion uniquely predicts group‑affiliation seeking.

Products

The products will include a correlation heatmap showing associations among narcissism, extraversion, group‑affiliation, and status seeking, two scatterplots with smooth lines depicting the bivariate relationships of narcissism with group‑affiliation and status seeking and a pair of bar or coefficient plots summarizing the hierarchical regression results, highlighting that extraversion uniquely predicts group‑affiliation while narcissism uniquely predicts status seeking when both are entered together.

vars <- narc_cor %>%
  select(narcissism, extraversion, group_aff, status)

cor_obj <- psych::corr.test(vars, use = "pairwise")
cor_mat <- cor_obj$r

cor_df <- cor_mat %>%
  as.data.frame() %>%
  tibble::rownames_to_column("var1") %>%
  pivot_longer(-var1, names_to = "var2", values_to = "r")
ggplot(cor_df, aes(x = var1, y = var2, fill = r)) +
  geom_tile(color = "white") +
  scale_fill_gradient2(low = "#2166ac", mid = "white", high = "#b2182b",
                       midpoint = 0, limits = c(-1, 1), name = "r") +
  geom_text(aes(label = sprintf("%.2f", r)), size = 3) +
  coord_fixed() +
  labs(x = NULL, y = NULL,
       title = "Correlations among narcissism, extraversion, and social motives") +
  theme_minimal(base_size = 12) +
  theme(axis.text.x = element_text(angle = 45, hjust = 1),
        panel.grid = element_blank())

Interpretation: From this heatmap, it can be interpreted that narcissism travels most strongly with status seeking, and extraversion travels most strongly with group‑affiliation.

library(broom)

m1_group <- lm(group_aff ~ extraversion, data = narc_cor)
m2_group <- lm(group_aff ~ extraversion + narcissism, data = narc_cor)


m1_status <- lm(status ~ extraversion, data = narc_cor)
m2_status <- lm(status ~ extraversion + narcissism, data = narc_cor)

coef_group <- tidy(m2_group, conf.int = TRUE) %>%
  filter(term != "(Intercept)") %>%
  mutate(
    outcome = "Group affiliation",
    term = recode(term,
                  extraversion = "Extraversion",

  )

coef_status <- tidy(m2_status, conf.int = TRUE) %>%
  filter(term != "(Intercept)") %>%
  mutate(
    outcome = "Status seeking",
    term = recode(term,
                  extraversion = "Extraversion",
                  narcissism   = "Narcissism")
  )

coef_all <- bind_rows(coef_group, coef_status)
ggplot(coef_all,
       aes(x = estimate, y = term,
           xmin = conf.low, xmax = conf.high,
           color = term)) +
  geom_vline(xintercept = 0, linetype = 2, color = "gray") +
  geom_point(size = 2) +
  geom_errorbarh(height = 0.15, size = 0.9) +
  facet_wrap(~ outcome) +
  scale_color_brewer(palette = "Dark2", name = "Predictor") +
  labs(x = "Unstandardized regression coefficient",
       y = NULL,
       title = "Unique effects of narcissism and extraversion on social motives") +
